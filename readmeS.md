SentraLock v1.0

SentraLock is a modern encryption and decryption application built on .NET 9.0, designed to provide secure, reliable, and efficient data protection. It enables users to encrypt and decrypt sensitive information using strong cryptographic practices while maintaining a clean and intuitive workflow.

Features:
-Strong encryption and decryption capabilities
-Built on .NET 9.0
-Designed with security-first principles
-Lightweight and efficient execution
-Clear separation of encryption and decryption logic
-Suitable for personal, enterprise, or educational use

Technology Stack:
Framework: .NET 9.0
Language: C#
Platform: Windows (cross-platform potential depending on build configuration)

==GETTING STARTED==
Prerequisites:
-.NET 9.0 SDK https://dotnet.microsoft.com/
-A compatible operating system (Windows recommended)

Installation:
1. Clone the repository:
   git clone https://github.com/yourusername/SentraLock.git

2. Navigate to the project directory:
   cd SentraLock

3. Build the application:
   dotnet build

4. Run the application:
   dotnet run
